# BBTPE
The Tape is a premium web-based basketball trivia experience designed for die-hard fans of professional basketball analytics and history. Players analyze an escalating series of detailed text clues spanning six core categories—Background, Draft, Career, Hardware, Franchise, and The Tell—to identify historical and modern basketball greats (from Michael Jordan and Kobe Bryant to Nikola Jokić).

The game includes intelligent client-side text parsing that auto-detects years and eras within the data cards, dynamically embedding specialized timeline badges next to text context dynamically.

✨ Features
Interactive Clue Progression: Progressively unlock layered clues about a player's draft class, career accolades, franchises, and signature traits.

Era & Year Recognition Engine: An integrated MutationObserver scans the DOM to dynamically inject context-aware era badges based on inferred historical years.

Obfuscated Gameplay Engine: Safe from quick inspections, player profiles and data strings are securely compressed/packed directly within the front-end application.

Premium Aesthetic: Designed with a sleek, premium dark mode color palette utilizing Playfair Display, Space Mono, and DM Sans typefaces for a luxury digital media presentation.

Pure Single-File Architecture: Built with standalone React (v18) via CDN hooks, making the entire platform incredibly fast, self-contained, and simple to host.
